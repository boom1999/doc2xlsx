# doc2xlsx

*Demand：*
> 下发通知文件为`word`格式，将回收大量含内嵌表格的`docx`文件
> 
> 需汇总到`excel`并且根据`map.xlsx`映射添加其他字段内容

```markdown
├── README.md
├── docx
│   └── GX-SB-XXXX-NumName
│       └── GX-SB-XXXX-NumName展品汇总表.docx
│       └── 展品Num
│          └── 附件
├── map
│   └── map.xlsx 
├── source
│   └── docx2xlsx_sum.py 
└── xlsx
    └── out_docx2xl.xlsx
4 directories
```
